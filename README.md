## Api

The Api provides you with full access to the Authenticator.

```shell
npm i --save wga-api
```

A clean and simple API design.

```ts
switch (platform) {
  case 'react':
    setUser(user)
    break
  case 'vue':
    this.user = user;
    break
  case 'express':
    res.render('index', {
      title: 'Hey',
      message: 'Hello there!',
    })
    break
}
```
